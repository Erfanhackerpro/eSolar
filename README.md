# eSolar - وب‌سایت شرکت سازه‌های خورشیدی

این پروژه یک وب‌سایت کامل و حرفه‌ای برای شرکت eSolar است که با استفاده از HTML5، CSS3 و جاوااسکریپت خالص ساخته شده است.

## ویژگی‌ها
- طراحی لوکس و فوتوریستی با الهام از اپل، تسلا و اسپیس‌ایکس
- انیمیشن‌های روان و تعاملات پیشرفته
- بهینه‌سازی سئو و دسترسی‌پذیری
- کاملاً واکنش‌گرا (موبایل-اول)
- بدون استفاده از کتابخانه‌های خارجی

## ساختار پروژه
- index.html - صفحه اصلی
- about.html - درباره ما
- services.html - خدمات
- projects.html - پروژه‌ها
- contact.html - تماس با ما
- css/ - فایل‌های استایل
- js/ - اسکریپت‌ها
- images/ - تصاویر و آیکون‌ها

## راه‌اندازی
فقط کافی است فایل‌ها را روی یک سرور وب قرار دهید یا با Live Server اجرا کنید.

## نکته
تصاویر نمونه (webp) باید توسط شما تهیه شوند. آیکون‌ها نیز قابل سفارشی‌سازی هستند.

## لایسنس
تمامی حقوق متعلق به eSolar می‌باشد.
